# Chigsaw
Chigsaw is a game of exploring and killing the BOSS!
====================================================
INSTALLATION:
1.Install python 2.7.x
2.Download game ZIP
3.Unzip the game
4.Launch file "start.pyc". On linux, open BASH, type the following command:
cd /home/USERNAME/GAME DIR NAME/, then type "python start.py"
5.Have fun!
